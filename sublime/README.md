###常用插件

1.Package Control，包管理器

2.ctags，函数跳转，跳转到定义

3.Alignment，'='对齐，快捷键Ctrl+Alt+Shift+A

4.Bracket Highlight，'{}'等高亮匹配

5.jQuery，jQuery提示

6.AutoFileName，自动完成文件名的输入

7.ConvertToUTF8，文件转码成utf-8

8.DocBlockr，可以自动生成PHPDoc风格的注释。它支持的语言有Javascript, PHP, ActionScript, CoffeeScript, Java, Objective C, C, C++。输入'/**'+enter

9.Terminal，打开命令行窗口，快捷键Ctrl+Shift+T

10.SFTP，代码映射到远程服务器
